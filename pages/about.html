О проекте WeatherAI

WeatherAI — это современное приложение для отслеживания погоды, которое использует передовые технологии и искусственный интеллект для предоставления точных данных и персонализированных рекомендаций.

Наша миссия

Мы стремимся сделать информацию о погоде не только точной, но и понятной, полезной и красивой. Наша цель — помочь вам планировать свой день, одеваться по погоде и быть готовыми к любым природным сюрпризам.

Функционал

  • Точный прогноз: Текущая погода и прогноз на 7 дней.
  • ИИ Анализ: Умные рекомендации по одежде и необходимости зонта.
  • Сравнение: Возможность сравнить погоду в двух городах.
  • Детализация: Данные о влажности, ветре, давлении, UV-индексе и др.
  • Визуализация: Интерактивные графики и карты.

Технологии

Проект построен с использованием современных веб-технологий:

  • HTML5 & CSS3 (Custom Properties, Flexbox, Grid)
  • Vanilla JavaScript (ES6+)
  • Open-Meteo API для данных погоды
  • Chart.js для графиков
  • Leaflet для карт
  • Lucide Icons для иконографии
