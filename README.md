# 🌤️ WeatherAI — Интеллектуальная метеоплатформа будущего

**WeatherAI** — это передовое веб-приложение для отслеживания погоды, которое объединяет высочайшую точность метеорологических сервисов с аналитической мощью искусственного интеллекта. Проект создан для обеспечения вашего максимального комфорта при планировании дня, поездок и активностей.

## ✨ Ключевые возможности

- 🎯 **Высокоточный прогноз:** Детальная информация о текущей погоде, почасовой прогноз на 24 часа и подробный обзор на 7 дней вперед.
- 🤖 **Продвинутый ИИ-Ассистент:** 
  - **Умный чат:** Задавайте вопросы ИИ голосом или текстом. ИИ самостоятельно анализирует ваш запрос и извлекает подробные данные о погоде для любого названного города в реальном времени.
  - **Помощник текстов:** Удобные списки быстрых подсказок (Quick Prompts) для мгновенного получения ответов в один клик.
- 🌍 **Прогноз городов Казахстана:** Выделенная вкладка с синхронным и параллельным мониторингом погоды во всех областных центрах и крупных городах РК.
- 🗺️ **Интерактивная метеокарта:** Визуализация в реальном времени слоев осадков, снега, направления ветра, температурных колебаний и облачности.
- 🌙 **Специализированные сервисы:** Встроенный 7-дневный лунный календарь, а также гороскоп и календарь садовода.
- 📊 **Экспорт и Аналитика:** Гибкий конструктор экспорта погодной истории и прогнозов в удобные для вас форматы (JSON, CSV, Excel).
- 📲 **Telegram Бот:** Быстрый кроссплатформенный доступ к прогнозам погоды прямо в вашем любимом мессенджере ([@WeatherAI_v2_bot](https://t.me/WeatherAI_v2_bot)).
- 🌗 **Персонализация и Локализация:** Поддержка трех языков (RU, KK, EN) и полная адаптация под светлую или темную тему операционной системы.

## 🛠 Технологический стек

Проект построен на современных веб-технологиях без использования тяжелых фреймворков для обеспечения максимальной скорости и легковесности:

- **Ядро:** HTML5, Vanilla JavaScript (ES6+), Web Speech API.
- **Стилизация:** CSS3 (Flexbox, Grid, CSS-переменные).
- **Интеграции (APIs):**
  - [Open-Meteo](https://open-meteo.com/) — поставщик высокоточных и бесплатных метеорологических данных.
  - [OpenWeatherMap](https://openweathermap.org/) — геокодирование, обратный поиск и координаты городов.
  - [RainViewer](https://www.rainviewer.com/) — радарные данные и слои осадков для карты.
- **Библиотеки:**
  - [Chart.js](https://www.chartjs.org/) — отрисовка гладких графиков температурных трендов.
  - [Leaflet](https://leafletjs.com/) — мощный движок для интерактивных карт.
  - [Lucide Icons](https://lucide.dev/) — красивая и современная SVG-иконография.

## 🏁 Быстрый старт

Проект архитектурно независим от серверного окружения (Static Web App). Вам не нужно настраивать базы данных или бэкенд:

1. Склонируйте репозиторий (или скачайте исходный код).
2. Запустите `index.html` с помощью локального веб-сервера.
   - *Рекомендуется использовать расширение **Live Server** для VS Code, чтобы избежать стандартных ограничений браузера (CORS) при загрузке локальных JSON файлов переводов.*
