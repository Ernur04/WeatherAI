# WeatherAI — Умный прогноз погоды с ИИ

WeatherAI — это современное веб-приложение для отслеживания погоды, которое сочетает в себе точность традиционных метеорологических данных и аналитическую мощь искусственного интеллекта. Проект создан для обеспечения максимального комфорта пользователей при планировании своего дня.

## 🚀 Основные возможности

- **Точный прогноз:** Текущая погода и детальный прогноз на 7 дней.
- **Почасовой прогноз:** Детальная информация об изменениях погоды в течение ближайших 24 часов.
- **ИИ-рекомендации:** Персонализированные советы по одежде и необходимости брать зонт на основе текущих условий.
- **Интерактивная карта:** Визуализация осадков, ветра, температуры и облачности в реальном времени.
- **Сравнение городов:** Удобный инструмент для сравнения погодных условий в двух разных локациях.
- **Многоязычность:** Поддержка русского (RU), казахского (KK) и английского (EN) языков.
- **Темная и светлая темы:** Интерфейс адаптируется под ваши предпочтения.
- **Дополнительные сервисы:** Лунный календарь, гороскоп и календарь садовода.

## 🛠 Технологический стек

- **Core:** HTML5, Vanilla JavaScript (ES6+).
- **Styling:** CSS3 (Flexbox, Grid, Custom Properties).
- **APIs:**
  - [Open-Meteo](https://open-meteo.com/) — основные погодные данные.
  - [OpenWeatherMap](https://openweathermap.org/) — геокодирование и поиск городов.
  - [RainViewer](https://www.rainviewer.com/) — слои осадков для карты.
- **Libraries:**
  - [Chart.js](https://www.chartjs.org/) — визуализация температурных трендов.
  - [Leaflet](https://leafletjs.com/) — интерактивная карта.
  - [Lucide Icons](https://lucide.dev/) — современная иконография.

## 📂 Структура проекта

- `index.html` — главная страница приложения.
- `js/` — логика приложения (работа с API, рендеринг, интерактив).
- `css/` — стилизация (модульная структура CSS).
- `locales/` — файлы переводов (JSON).
- `pages/` — дополнительные страницы (о проекте, политики, доп. сервисы).
- `img/` — графические ресурсы.

## 🏁 Как запустить

Данный проект является статическим веб-приложением и не требует сложной серверной настройки.

1. Клонируйте репозиторий или скачайте архив с кодом.
2. Откройте файл `index.html` в вашем браузере.
   - _Рекомендуется использовать локальный сервер (например, расширение **Live Server** для VS Code) для корректной работы всех функций и путей._
